# Theatre Management System

A console-based theatre management system built in C++ that allows 
users to book tickets for movies, plays, and concerts through an 
interactive menu-driven interface.

## About
Built during my first semester as part of my Programming Fundamentals 
course. This project was more advanced than my previous work — it 
introduced me to structures, functions, and building a complete 
user-facing application in C++.

## Features
- User login with name and CNIC validation
- Browse and book Movies, Plays, and Concerts
- Genre-based filtering for shows
- Interactive seat selection with Premium and Standard options
- Snack purchasing system with 7% GST calculation
- Loyalty points system based on total spending
- Printable booking receipt
- Customer feedback collection

## What I Learned
- Structs and passing by reference in C++
- Multi-dimensional arrays for seat management
- Input validation and error handling
- Breaking a large problem into functions
- Building a real user experience in console

## Tech Stack
- Language: C++
- IDE: Dev-C++ / Code::Blocks

## How to Run
1. Clone the repository
2. Open the `.cpp` file in your IDE
3. Compile and run
